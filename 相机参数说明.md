# 相机参数配置规范 (Camera Configuration Profile)

## 1. 基础采集设置 (Basic Features)
| 参数项 | 配置值 | 备注 |
| :--- | :--- | :--- |
| **帧率控制使能** | **Off (关闭)** | 释放硬件最高性能 |
| **自动曝光 (Exposure Auto)** | **Off (关闭)** | 竞赛环境建议手动锁定，防止画面闪烁 |
| **曝光时间 (Exposure Time)** | **3800 $\mu s$** | 对应约 250fps (理论上限 4000 $\mu s$) |
| **增益自动 (Gain Auto)** | **Continuous (连续)** | 根据环境光线自动补偿亮度 |
| **像素格式 (Pixel Format)** | **Bayer RG 8** | 原始数据格式，减少传输带宽压力 |

## 2. 图像处理 (ISP & Algorithm)
* **亮度 (Brightness):** `50` (标准中值)
* **白平衡 (Balance White Auto):** `Continuous` (自动校正环境色偏)

## 3. 传输与触发控制 (Transport & Trigger)
* **AOI / ROI:** `None` (全分辨率输出)
* **触发模式 (Trigger Mode):** `Off` (连续采集模式)
* **用户设置保存 (User Set Control):** `User Set 1` (已固化至相机内部)

## 4. 镜头物理调节 (Lens Physical)
* **光圈 (Aperture):** `F2.4`
* **对焦 (Focus):** `Manual` (手动对焦，建议针对靶标距离加固对焦环)

---

## 5. 进阶功能补充 (Advanced Features Summary)

### 5.1 超级 Bayer (Super Bayer)
* **状态：** 当前未开启 (`Disabled`)
* **作用：** 海康专有 FPGA 级去噪技术。在不改变 Bayer RG 8 数据格式的前提下，在硬件层提升画质，降低暗部噪点。

### 5.2 色彩校正矩阵 (CCM)
* **作用：** 通过 $3 \times 3$ 矩阵进行颜色校准，使色彩更接近人眼真实视觉。
* **注意：** 在输出 Bayer 格式时，开启此项通常会消耗 RK3588 端 MVS SDK 的软件算力。

### 5.3 超级调色板 (Super Palette)
* **作用：** 类似于 HSL 调节，允许针对特定颜色（如红/绿色块）进行色相与饱和度的精细微调。在复杂的颜色识别任务中，可用于增强目标特征色。
